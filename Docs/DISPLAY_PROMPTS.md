# Display and design prompts

Use these prompts to generate visual materials for the WCS Apple ecosystem.

## Memory Atlas app display

High-fidelity iPhone/iPad product mockup for Memory Atlas, an Apple-native dementia-care reminiscence app. Calm therapeutic interface, warm off-white cards, sage green and soft gold accents, large accessible typography, caregiver dashboard, memory profile, artifact cards with photos/audio/story tags, guided session player with calm prompt text and gentle timer. The mood should feel safe, premium, clinical but human, Apple-native, and easy for older adults and carers.

## ScholarSphere Studio app display

High-fidelity iPad creator workspace for ScholarSphere Studio. Show a modular lesson builder with spatial scene blocks, media assets, prompts, quizzes, publish status, and analytics. Elegant Apple productivity style, clean panels, soft depth, rich but restrained education palette, Pencil-friendly layout.

## Presence Play app display

High-fidelity Vision Pro spatial storytelling concept. Show a shared therapeutic narrative lobby and a calm immersive scene preview with two participants, facilitator controls, branching choices, reflection prompt, and spatial audio cues. Premium Apple spatial computing look, translucent panels, soft environment, relationship-centered care tone.

## Architecture diagram prompt

Create a clean architecture diagram titled “Memory Atlas System Architecture”. Show client layer with SwiftUI app, WCSCore, WidgetKit/Live Activities, Future visionOS. Show API layer `/api/v1`. Show Supabase Postgres, Supabase Storage, and CloudKit private database. Label Supabase as system of record, CloudKit as private mirror/cache, and storage as private bucket with signed URLs. Use calm WCS colors: navy, sage, cream, gold.
