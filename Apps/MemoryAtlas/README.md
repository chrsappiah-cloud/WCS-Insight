# Memory Atlas

First WCS implementation target: an Apple-native dementia-care reminiscence app for families, carers, and clinicians.

## MVP surfaces

- Caregiver dashboard
- Person memory profile
- Memory artifacts: photo, audio, video, story
- Guided session builder
- Guided session player
- Calm mode
- WidgetKit “Today’s memory” boundary
- ActivityKit active session boundary
- CloudKit personal mirror boundary
- Future RealityKit memory rooms
